# JS Build Template

Uses

- broccoli (building)
  - browserify (modules)
  - babel (es6 to es5)
- jest (testing)
- react

## Initialization

`npm install`

## Building

### Development

`broccoli build out`

### Production

`BROCCOLI_ENV=production broccoli build production`

## Running

`broccoli serve`

## Testing

`npm test`
